## Packages
framer-motion | For smooth page transitions and micro-interactions
lucide-react | For beautiful icons
clsx | For conditional class names (often used with tailwind-merge)
tailwind-merge | For merging tailwind classes safely

## Notes
- The app uses `wouter` for routing.
- Audio integration is provided in `client/replit_integrations/audio`.
- Text adventure uses `client/replit_integrations/audio/useVoiceStream` for voice interaction.
- Images for the text adventure are generated via `/api/generate-image`.
